Features

Everything you need,
nothing you don't.

No bloat. No accounts. No telemetry. Just raw speed and power, right in your browser.

⚡

Instant Launch

Opens in under 50ms. Fuzzy search results appear as you type. Zero startup cost thanks to lazy initialization.

🔍

Fuzzy Search Everything

Powered by quickfuzz — 7x faster than Fuse.js. Search tabs, bookmarks, history, commands, and more with match highlighting.

🛠

Tab Power Tools

Pin, mute, move, close, duplicate, suspend, group tabs. Multi-select with Space for batch operations.

✉

Text Snippets

Create reusable text snippets with dynamic variables. Type a trigger keyword to expand. Supports date, URL, title placeholders.

🔗

Aliases & Quicklinks

Type "mail" to open Gmail. Type "jira PROJ-123" to jump to a ticket. Parameterized URLs with %s replacement.

🕵

Shadow DOM Isolation

Rendered in a closed Shadow DOM. Page CSS can't touch it. Hatch CSS can't leak out. Works perfectly on every website.

📊

Frecency Ranking

Commands you use frequently and recently bubble to the top. Your workflow gets faster the more you use Hatch.

🌐

18 Search Engines

Type "gh react" to search GitHub. "yt lofi" for YouTube. "mdn fetch" for MDN. Add your own custom engines.

🔒

100% Local

No servers. No databases. No accounts. Everything stored in chrome.storage.local. Your data never leaves your machine.
